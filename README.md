#**Galaxy Generator (WinForms GUI)**

Графический интерфейс для визуализации моделей галактик с расширенными настройками.

Основные возможности:

Выбор типов: Переключение между спиральной, эллиптической и миндалевидной моделями.
Настройка параметров: Пользователь может задавать размер поля (минимум 200x200) и параметры генерации.
Визуальные эффекты:
Динамическое изменение цвета звезд от центра к краям.
Плавное изменение прозрачности звезд в зависимости от плотности.
Работа с фоном: Поддержка загрузки BMP-файлов в качестве подложки для изображения галактики.
